# AI Content Detector

Detect whether your content was written by AI. A web application that analyzes text or uploaded documents to determine if they were likely generated by AI models like GPT-3, GPT-4, Claude, or Gemini.

## Features

- **Text Input**: Paste text directly for analysis
- **File Upload**: Support for `.txt`, `.docx`, and `.pdf` files (max 10MB)
- **AI Detection**: Get overall AI probability with visual indicators
- **Model Analysis**: View detection probabilities across multiple AI models
- **Sentence Highlighting**: See which sentences are likely AI-generated vs human-written
- **PDF Reports**: Download detailed analysis reports

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **PDF Generation**: jsPDF
- **File Parsing**: mammoth (DOCX), pdf-parse (PDF)

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ai-content-detector
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

### Text Detection

1. Navigate to the homepage
2. Paste or type your text into the input area
3. Click "Detect AI Content"
4. View your results on the results page

### File Upload

1. Navigate to the homepage
2. Click the upload area or drag and drop a file
3. Supported formats: `.txt`, `.docx`, `.pdf`
4. Click "Detect AI Content"
5. View your results on the results page

### PDF Report

- On the results page, click "Download PDF Report"
- The report includes all detection details and analysis

## API Endpoint

### POST `/api/detect`

Detects AI-generated content from text or file.

**Request (JSON):**
```json
{
  "text": "Your text content here..."
}
```

**Request (FormData - file upload):**
- `file`: File object (.txt, .docx, .pdf)

**Response:**
```json
{
  "success": true,
  "data": {
    "overallProbability": 75,
    "isAI": true,
    "sentences": [
      {
        "text": "Sentence text",
        "isAI": true,
        "probability": 82
      }
    ],
    "models": {
      "gpt3": 75,
      "gpt4": 80,
      "claude": 68,
      "gemini": 72
    }
  }
}
```

## Replacing the Mock Detection Engine

The current detection engine (`lib/detectionEngine.ts`) uses heuristics to simulate AI detection. To replace with a real ML model:

1. **Option A - External API**:
   - Modify `app/api/detect/route.ts` to call your AI detection API
   - Update the response parsing to match your API's format

2. **Option B - Local Model**:
   - Integrate a local ML model (e.g., using TensorFlow.js or ONNX runtime)
   - Modify `lib/detectionEngine.ts` to load and run your model

Example replacement for external API:

```typescript
// lib/detectionEngine.ts
export async function detectAIContent(text: string): Promise<DetectionResult> {
  const response = await fetch('YOUR_API_ENDPOINT', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ text }),
  });

  const data = await response.json();
  return {
    overallProbability: data.ai_probability,
    isAI: data.ai_probability > 50,
    sentences: data.sentences,
    models: data.models,
  };
}
```
